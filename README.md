# 🌟 Review Tokens Smart Contract

## 📌 Project Description

Review Tokens is a decentralized smart contract built on the Stellar Soroban platform that rewards users with tokens for submitting reviews. The contract incentivizes user engagement and honest feedback by providing token-based rewards for each review submitted on-chain.

## ⚙️ What It Does

* Allows users to submit reviews through the smart contract
* Automatically rewards users with tokens for each review
* Maintains a record of:

  * Total reviews submitted by each user
  * Token balances of users

---

## 🚀 Features

* 📝 On-chain review submission
* 💰 Token rewards system (fixed reward per review)
* 👤 Per-user review tracking
* 📊 Token balance lookup
* ⚡ Lightweight and efficient Soroban implementation
* 🔐 Fully decentralized logic (no central authority)

## 📦 Contract Functions

### submit_review(user, review)

Submits a review and rewards the user with tokens.

### get_balance(user)

Returns the token balance of the specified user.

### get_reviews(user)

Returns the total number of reviews submitted by the user.
